For a given year, each csv dataset contains:

exporter = ISO 3 letters exporting country (in FGO 2022, this variable is "i")

importer = ISO 3 letters importing country (in FGO 2022, this variable is "j")

hs6 = HS6 code, revision 2007

year = year (2001, 2004, 2007, 2010, 2013, 2016, 2019)

adv = ad valorem equivalent of the preferential applied tariff (source: MAcMap-HS6, from ITC-CEPII)

trade = trade value in current USD (source: BACI database, from CEPII)

A mapping between ISO code and country names is provided (countries.csv)

Please quote
Fontagné, L., Guimbard, H. and Orefice, G. (2022) Tariff-Based Product-Level Trade Elasticities. Journal of International Economics, 2022, vol. 137
